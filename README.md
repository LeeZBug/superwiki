# 🛠️ SuperWiki

> **记录踩过的坑，填平未来的路。**
> 本仓库用于收集、整理和归档在开发、运维及日常工作中遇到的各类疑难问题与解决方案，使用技巧等，旨在构建一个可检索、可复用的个人/团队知识库。

## 📖 关于本仓库

在日常技术工作中，我们经常会遇到各种诡异的报错、隐蔽的 Bug 以及复杂的配置问题。这些问题往往在解决后容易被遗忘，当下次再遇到时又需要重新排查。

本仓库的核心价值在于：
-   **知识沉淀**：将零散的排查经验转化为结构化的文档。
-   **快速检索**：通过标准化的标签和目录，实现秒级定位解决方案。
-   **避免重复造轮子**：减少团队成员在相同问题上的时间浪费。
-   **新人赋能**：作为 Onboarding 资料，帮助新成员快速了解系统历史与常见陷阱。
